Здравствуйте!!! Вас приветствует программа "Калькулятор площадей". Она позволяет рассчитать площадь
выбранной фигуры по указанным параметрам. Также поддерживается загрузка списка фигур с нужными
параметрами из XML-файла и сохранение результата расчета в TXT-файл.

Всего поддерживается 5 фигур:
--Прямоугольник(нужно ввести две стороны)
--Круг(нужно ввести радиус)
--Эллипс(нужно ввести 2 полуоси)
--Треугольник(нужно ввести 3 стороны)
--Трапеция(нужно ввести 2 стороны и высоту)


1)НАСТРОЙКА И ЗАПУСК
В поставку данной программы входит:
--README.txt - данный файл
--example.xml - XML-файл с примерами входных параметров
--/lib - папка с необходимыми для работы программы библиотеками
--square-1.1.jar - исполняемый jar-файл

Для начала работы нужно разархивировать программу. Дальше через консоль в разархивированной
папке выполнить команду(необходимо установить jdk 1.8):
java -jar square-1.1.jar


2)РАБОТА С ПРОГРАММОЙ

В отткрывшемся диалоговом окне будет предложено выбрать систему ввода(через консоль или через
XML-файл)и систему вывода(только в консоль или с сохранением с тектовый файл).

Если был выбран ввод через XML-файл нужно будет указать путь к данному файлу(либо из текущей
директории, либо полный).

Если был выбрано сохранение в TXT-файл, то он файл появится в разархивированной папке под 
название result.txt(при каждом заходе в стартовое меню файл очищается).
